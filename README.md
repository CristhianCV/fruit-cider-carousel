# Fruit Cider 🍐🍏🥝

> Desing: [Figma Tutorial: 3D Carousel Slider](https://www.youtube.com/watch?v=XKSNDxp-I14)

Carousel sliders with rotating effects

Made with React + Tailwind + GSAP

## Get Started

1. Install the dependencies

```
cd fruit-cider-carousel
npm install
```

2. Run app

```
npm run dev
```
